# 🧪 CDB - Centro de Diagnósticos Biológicos

[![Laravel](https://img.shields.io/badge/Laravel-Vite-red.svg)](https://laravel.com/)
[![HealthTech](https://img.shields.io/badge/Dom%C3%ADnio-Laboratorial-blue.svg)]()

Sistema especializado para gestão de laboratórios e centros de diagnóstico, focando no fluxo completo desde a requisição até a entrega do laudo.

## 🚀 Funcionalidades
- Gestão de pedidos laboratoriais.
- Interface moderna integrada com Vite.
- Controle de fluxos de exames e resultados.

---

## 🚦 Início Rápido
1. `composer install`
2. `npm install && npm run dev`
3. `php artisan serve`
